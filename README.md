## Local Setup Instructions

To run this project on your local machine, follow the steps below:

1. Clone the repository  
Open your terminal and run:  
_git clone https://github.com/UBSITStaMariaCMiaKyla/appdev1-sdg11-terminalguide.git_  
_cd your-repo-name_  

2. Install dependencies 
Make sure you are inside the project folder, then run:  
_npm install_  


3. Start the development server    
Run the following command:  
_ng serve_  

4. Open the application  
Once the server is running, open your browser and go to:  
http://localhost:4200/  

---

**Note:**    
Make sure Node.js and Angular CLI are installed on your system. If Angular CLI is not installed yet, you can install it using:  
_npm install -g @angular/cli_
